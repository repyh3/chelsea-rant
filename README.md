# Chelsea Rant Widget

A chaotic GitHub README widget for Chelsea fans. Shows live match scores and provides a random rant based on the outcome.

## Usage

Add this to your Profile README:

```markdown
![Chelsea Rant Widget](https://chelsea-rant.vercel.app/api/rant)
```

## Features

- **Live Data**: Scores and opponents fetched from the official FPL API.
- **Dynamic Rants**: Rotates through a library of fan rants appropriate for wins, draws, or losses.
